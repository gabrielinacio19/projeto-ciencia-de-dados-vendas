# Análise de Vendas - Ciência de Dados

Projeto acadêmico de Ciência de Dados com foco em análise descritiva, preditiva e prescritiva usando a base `data/sales_data.csv`.

## Como executar

1. Ative o ambiente virtual.
2. Instale as dependências:

```powershell
pip install -r requirements.txt
```

3. Execute o projeto:

```powershell
python main.py
```

Os gráficos são gerados na pasta `output/`.
